# Threaded Comments System

Course project: Threaded Comments System (similar to Reddit).

## Stack

- C#
- ASP.NET Core Web API
- PostgreSQL
- Entity Framework Core
- Vue.js

## Architecture

Clean Architecture:

- Domain
- Application
- Infrastructure
- WebApi

## Features

- threaded comments
- replies
- reactions (like / dislike)
- notifications
- comment tree traversal
